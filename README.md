# NumberInfo - Phone Number Intelligence Tool

## ⚠️ CRITICAL LEGAL NOTICE ⚠️

**This tool provides ONLY publicly available information about phone numbers.**

It does NOT and CANNOT:
- Access IMEI numbers
- Retrieve call logs
- Access contacts
- Track locations
- Obtain personal user information

**Attempting to access such data without authorization is ILLEGAL and violates:**
- Computer Fraud and Abuse Act (CFAA)
- Electronic Communications Privacy Act (ECPA)
- GDPR (EU)
- POPIA (South Africa)
- Similar privacy laws worldwide

## Purpose

This educational tool demonstrates:
- Phone number validation
- Country code parsing
- Carrier detection
- Geographic information (public data only)
